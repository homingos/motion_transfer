🎬 Motion Transfer Demo

Test both Dev (R2 Mode) and Feat (Direct Upload) environments

Dev Environment

R2 MODE - Avatar ID
https://ai-team-flam-dev--motion-transfer-dev.modal.run
How it works: You provide an avatar ID, the system finds the image in R2 storage, generates the motion, and saves it back to R2. Perfect for production workflows with existing avatars.
💡 Tip: After submitting, you'll get a request_id. Use the "Check Status" tab to monitor progress. Takes 2-5 minutes.
💡 Info: Check if avatar exists, where the image is stored, and if animation is ready.

Feat Environment

DIRECT MODE - Image Upload
https://ai-team-flam-feat--motion-transfer-feat.modal.run
How it works: Upload an image directly and get the MP4 video back immediately (waits for processing). Great for quick testing and demos without needing a pre-existing avatar.
💡 Tip: Upload your image and wait 2-5 minutes. The system will process and return your MP4 video directly.
💡 Tip: Same as Dev environment. Async generation using avatar ID.
